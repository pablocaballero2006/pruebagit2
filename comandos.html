Comandos básicos en Git Bash (tipo Unix)

PS

Muestra los procesos que se están ejecutando en el sistema.

Ejemplo:

ps

GREP

Busca texto dentro de archivos o resultados de otros comandos.

Ejemplo:

cat archivo.txt | grep "palabra"

SORT

Ordena líneas de texto alfabéticamente.

Ejemplo:

sort archivo.txt

CUT

Extrae partes específicas de cada línea (columnas).

Ejemplo:

cut -d "," -f 1 archivo.csv

HEAD

Muestra las primeras líneas de un archivo.

Ejemplo:

head archivo.txt

TAIL

Muestra las últimas líneas de un archivo.

Ejemplo:

tail archivo.txt
